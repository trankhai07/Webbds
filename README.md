1. Đề tài: Web Sàn giao dịch bất động sản
2. Các công nghệ sử dụng: 
   + Front-end: HTML, CSS, JS
   + Back-end: Java(Servlet)
   + Database: MySQL
3. Một số lưu ý khi code front-end:
   + Dùng chung thư viện CSS
   + Đặt tên theo tiêu chuẩn BEM
4. Chức năng:
   + Tìm kiếm 
   + Đăng nhập, đăng ký
   + Đăng tin
   + CRUD trang admin
